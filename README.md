# imageViewer

## Manual Test Notes

- **Tab drag reorder**
	- 開いているタブをドラッグして順番が変更されるか確認
	- 右クリックメニューやタブ閉じるボタンを操作してもドラッグが誤発火しないこと
	- ドラッグ後に該当タブが選択されたままであること
	- アプリを終了→再起動しても並び順が保持されること

- **Visual swap animation**
	- タブをドラッグするとヘッダーがマウス位置に追従し続けること
	- 隣接タブの中央を越えた瞬間にタブが入れ替わり、押しのけられたタブが滑らかに戻ること
	- ドラッグを離した際にタブが最終位置へスムーズにスナップすること
